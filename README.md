- Create a new Repository from the internet (Használhatod a URL : https://github.com/LukaPollo/Bario.git)
### Majd menj be Unity-be : 
- Add
- Add project from disk
- Válasz ki a GitHub Desktop file amit leclone-oltunk.
- Inditsd el
- Csinálj amit akarsz unityba.
### Hogyan mentsd vissza a repoba:
- Mentsd el az összes fáljt
- Lépj ki a Unity Editorból.
- Mivel a kinyitott folder a Unity Hubban a Githubos Clone-olt folder nem kell átmasolgatni őket
- Csak szimplán ird le, hogy mit csináltál -> commit -> push origin
